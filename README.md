# APP_DEV_SQUAD - Android

Projeto convertido para módulo Android com Jetpack Compose e Hilt.

Como rodar:
1. Abra um terminal na raiz do projeto.
2. Se você não tem o wrapper, gere com: `gradle wrapper` (ou instale Gradle localmente).
3. Rode: `./gradlew assembleDebug` (no Windows Powershell: `./gradlew.bat assembleDebug`).
4. Para instalar no dispositivo: `./gradlew installDebug`.
5. Ver logs: `adb logcat`.

Requisitos: JDK 11+, Android SDK, emulador ou dispositivo conectado.
